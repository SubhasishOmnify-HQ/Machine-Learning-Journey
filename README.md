# 🤖 Machine Learning

## 📖 Overview

Machine Learning (ML) is a branch of Artificial Intelligence (AI) that enables computers to learn patterns from data and make predictions or decisions without being explicitly programmed. Instead of following predefined rules, machine learning models improve their performance by learning from historical data.

---

## 🎯 Objectives

* Learn patterns from data
* Make accurate predictions
* Automate decision-making
* Solve real-world problems
* Continuously improve with new data

---

## 🧠 Types of Machine Learning

### 1. Supervised Learning

Uses labeled data to train models.

**Examples**

* House Price Prediction
* Spam Email Detection
* Heart Disease Prediction

**Algorithms**

* Linear Regression
* Logistic Regression
* Decision Tree
* Random Forest
* Support Vector Machine (SVM)
* K-Nearest Neighbors (KNN)
* Naive Bayes

---

### 2. Unsupervised Learning

Uses unlabeled data to discover hidden patterns.

**Examples**

* Customer Segmentation
* Market Basket Analysis

**Algorithms**

* K-Means Clustering
* DBSCAN
* Hierarchical Clustering
* Principal Component Analysis (PCA)

---

### 3. Semi-Supervised Learning

Uses a combination of labeled and unlabeled data.

---

### 4. Reinforcement Learning

An agent learns by interacting with an environment through rewards and penalties.

**Applications**

* Robotics
* Self-driving Cars
* Game AI

## 🔁 Cross Validation

Cross Validation evaluates model performance using different subsets of the dataset, providing a more reliable estimate than a single train-test split.

## ⚖️ Overfitting vs Underfitting

### Overfitting

* Learns training data too well
* Poor performance on new data

### Underfitting

* Fails to learn important patterns
* Poor performance on both training and testing data

## 📌 Best Practices

* Understand the problem before selecting an algorithm.
* Perform Exploratory Data Analysis (EDA).
* Clean and preprocess data carefully.
* Split data properly.
* Scale features when necessary.
* Use appropriate evaluation metrics.
* Apply Cross Validation.
* Tune hyperparameters for better performance.
* Save trained models for reuse.
* Deploy and monitor models in production.
